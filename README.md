**上位机信号编码**

上位机上输入的数据为float型，将其扩大100倍后转换为int型然后将int转换为两个字节的数据发送，接收端再进行解码，其发送数据格式如下;

| SOF(1byte) |         data(28byte)         | EOF(1byte) |
| :--------: | :--------------------------: | :--------: |
|    0x0d    | data_0[7:0] ... data_27[7:0] |    0x0a    |
